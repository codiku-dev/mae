Welcome to Mia installer !
Mia is a simple AI assistant fully working offline with 0 data retention. Nothing is sent to a cloud or stuff like that :)


Duration : 10 minutes (you have to download the AI model which is 5gb )


You are about to install this :

+ Mia (300mb): The tool that allow you to chat with AI
+ Ollama (150mb): External tool that allow run an AI on your machine 
+ Llama3.1 (4.5gb): External AI model built by Facebook
